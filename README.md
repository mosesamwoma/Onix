# Onix — Phone Company Website

A multi-page website for a fictional phone company, built as a group project during our first year, first semester at Meru University — Bachelor of Science in Data Science. This project helped us practice HTML and CSS fundamentals while building a complete user-facing website.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)
- A text editor or IDE like VS Code (optional, for editing)

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/mosesamwoma/Onix.git
   cd Onix
```

2. Open `Home Page.html` in your browser

   **OR**

   Visit the live site: https://helpful-empanada-60d671.netlify.app/
